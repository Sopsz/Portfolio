Site Vanoplie

Application Web Fullstack

Description du projet

Cette application est le site de Vanoplie présentant leurs entreprise et leurs produits

Caractéristiques

Catégorie Détails
Technologies Joomla5
Durée 2 mois (2025)
Fonctionnalités clés
  • Créer un site professionnel qui présente l'entreprise Vanoplie.
  • Fournir un moyen à l'utilisateur de se connecter a un compte et d'envoyer un mail sur la messagerie de Vanoplie avec un module intégré dans le site.
Difficultés rencontrées
  • Adapter la production en fonction des besoins du maître de stage.
  • Respecter des delais.
Compétences acquises
  • Garder le maître de stage informé de l'avancé en vulgarisant.
  • Travailler en équipe sur un projet professionnel.

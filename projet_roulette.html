Site Roulette

Application Web Backend

Description du projet

Cette application web permet de créer un compte, se connecter puis de miser sur un nombre avec de la monnaie fictive.

Caractéristiques

Catégorie Détails
Technologies PHP, HTML, CSS
Durée 1 semaine (2023)
Fonctionnalités clés
  • Créer et se connecter à un compte
  • Jouer à un jeu de hasard avec de la monnaie fictive
Difficultés rencontrées
  • Intégrer la création/connexion de compte
  • Contrer les failles XSS et injection SQL
Compétences acquises
  • Créer un site fonctionnel avec une arborescence type MVC
  • Coder soi-même le DAO & PDO en PHP

Site Transport

Application Web Fullstack

Description du projet

Cette application web permet à un professeur, via son compte, de gérer une classe en ajoutant des élèves et de leur attribuer des points en fonction du moyen de transport utilisé pour venir en cours.

Caractéristiques

Catégorie Détails
Technologies Laravel, Tailwind, MySQL
Durée 2 mois (2025)
Fonctionnalités clés
  • Se connecter avec un compte créé par l'administrateur
  • Gérer des élèves via un CRUD
  • Attribuer des points aux élèves en fonction de leur moyen de transport
Difficultés rencontrées
  • Réaliser une application web facile d'utilisation et agréable visuellement
Compétences acquises
  • Maîtriser les fonctionnalités de Laravel (Factories / Seeders / Routes / Blade / Contrôleur)

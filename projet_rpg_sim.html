RPG SIM

Mini-jeu RPG de combat

Description du projet

C'est un projet codé en Java qui permet de faire des combats entre plusieurs classes (mage, guerrier, prêtre) en tour par tour.

Caractéristiques

Catégorie Détails
Technologies Java, MySQL
Durée 1 mois (2024)
Fonctionnalités clés
  • Choisir sa classe et la classe de son adversaire
  • Possibilité d'utiliser plusieurs habilités différentes donnant un aspect stratégique
Difficultés rencontrées
  • Faire le lien entre le code et la base de données via un DAO et PDO (JDBC)
  • Adapter le jeu pour qu'il soit équilibré et plaisant à jouer
Compétences acquises
  • Apprendre l'héritage et le polymorphisme
  • Travailler sur un projet en équipe
  • Utiliser un outil de gestion de projets et respecter des délais
